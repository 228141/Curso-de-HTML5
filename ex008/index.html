Texto Deletado

Podemos marcar um texto como excluído, para indicar que ele deve ser lido, mas não deve ser considerado.

Texto inserido

Podemos marcar um texto como inserido para indicar que ele deve ser lido, mas não deve ser considerado.

Existe também o sublinhado com a tag U (não semântico).

Texto Sobrescrito

Para inserir no texto coisas do tipo, exemplo x20+3, usa-se a tag SUP.

Texto Subscrito

Para inserir no texto coisas do tipo, exemplo H20, usa-se a tag SUB.
