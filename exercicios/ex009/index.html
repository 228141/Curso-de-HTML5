Trabalhando com Listas

Listas Ordenadas

As listas ordenadas estão são utilizadas com a tag ol. E pode ser personalizado o acompanhado com a seleção do tipo "type", podendo ser selecionado cinco tipos (1, A, a, I, i) numerações, letras maiusculas, letras minusculas, numeral romanos em maiuscula ou numeral romanos em minuscula. Conforme os exemplos abaixo:

  1. Acordar
  2. Ligar para João
  3. Escovar os dentes

  3. Escovar os dentes

Para iniciar a lista com um número desejado a seleção do tipo "type", segue-se com "start" e o numeral desejado. Conforme os exemplos abaixo:

  1. Acordar
  2. Ligar para João
  3. Escovar os dentes

Lista de Definições

Abaixo estaremos explicando e utilizando as tags de definições.

Definition list (dl)
Toda lista de definições está dentro de uma tag "dl" (definition list)
Definition Term (dt)
Cada termo de definição está dentro de uma tag "dt" (definition term)
Definition Description (dd)
E cada termo de descrição está dentro de uma tag "dd" (definition description).
