Diretos Autorais.

Para o uso de imagens em site, deve se tomar cuidado pois existem imagens que são protegidas por direitos autorais, e na hora de utilizar as imagens é preciso utilizar as imagens com direitos de uso.

No Google imagem, você pode:

  1. Clicar em "ferramentas";
  2. Clicar em " Direitos de uso";
  3. Clicar em "Marcadas para reutilização" - essas são as imagens que podem ser usadas comercialmentes no sites disponibilizadas pelo Google;

Tratamento de Imagens.

Para utilizar o tratamento de imagens, utilizaremos o "GIMP", para fazer as tratativas das imagens baixadas.

Uma imagem é um conjunto de pontos, e fica ocupando muitos bytes, e quanto maior o arquivo, mais pesados fica o site e mais dificil de ser carregado.
